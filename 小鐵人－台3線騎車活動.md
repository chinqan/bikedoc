## 三、騎乘路線總覽

本次活動路線以 **龍潭大池** 為起終點，沿台3線及周邊山區道路騎乘，途中經過大北坑、內環山、國道三號路橋下、7-11 騰達門市、濟世宮、中油正新站等地點，最後返回龍潭大池。

全程約 **41 km**，路線具有山區起伏，並設有多個補給點、折返點與交通管制提醒點。

**本次活動分為 5 組 + 機動組 + 補給車，請依分組行進，詳見下方分組名單。**

活動重點：

- 安全完成騎乘
- 團體行進、互相照應
- 遵守交通規則
- 聽從交管與工作人員指示
- 不競速、不搶快

## 十、路線難度與騎乘提醒

本次活動總里程約 **41 km**，全程包含多段山區起伏及下坡路段，並非完全平路。

路線特色包括：

- 中長距離騎乘
- 山區道路起伏
- 多處叉路口
- 多個交通會合點
- 補給點間距約 6～11 km

本活動適合具備基本騎乘能力的參加者，包括：

- 能穩定騎乘 20 km 以上
- 具備基本變速能力
- 能控制上下坡速度
- 會保持安全車距
- 能遵守團騎規則與交管指示

---

## 十一、安全注意事項

| 項目 | 重點 |
|---|---|
| 🛡️ **安全第一** | 靠右騎乘、不逆向、不闖紅燈、不任意超車、不併排聊天影響交通、下坡不競速；遵守工作人員與交管人員指示 |
| 🚦 **叉路口／交管** | 路線含多處叉路與交管待轉：大北坑（1 km）、內環山（8.3 km）、補給點前後、折返點、加油站進出口、回程接近龍潭大池路段。<br>通過叉路口減速；補給車不易臨停的路段，請**聽從押隊鐵爸／鐵媽指示** |
| 🚧 **施工路段** | 領隊將提前廣播；全隊**減速慢行**、**單線依序通過**；必要時下車**牽車**；雨天路面濕滑須加倍謹慎。<br>押隊鐵爸／鐵媽會殿後，**確認全組通過後再前進** |
| ⛰️ **上坡** | 使用適當變速、不硬踩、保持穩定呼吸、與前車保持距離 |
| ⬇️ **下坡** | 雙手握穩煞車、不放手滑行、不超速、不任意超車、入彎前先減速；注意砂石、水溝蓋與路面坑洞 |
| 💧 **補水／體力** | 不要等到口渴才喝水：出發前先喝水、途中適量補水、每個補給點檢查水量。<br>若有頭暈、抽筋、噁心或體力不支，請**立即告知工作人員** |

## 十二、雨天備案

- 主辦單位於活動前一日（5/23）晚間 20:00 前依氣象預報決定是否照常舉行，並於 LINE 群組統一公告。
- 請所有參加者自備**騎行雨具**，以備不時之需。
- 施工路段雨天路況更差，領騎需特別注意控速，視情況決定是否牽車通過。
- 若遇大雨、路況不佳，**領隊將評估是否調整路線或部分路段棄賽**。
- **小鐵人精神：** 在可安全行進的情況下，盡力完成訓練挑戰！

## 十三、途中棄賽或身體不適處理

若參加者途中體力不支或機械故障，請：

1. 告知身旁隊員或工作人員
2. 移至路側安全處等候
3. 聯繫**補給車**（羊來了 / 金錢豹）接送返回終點
4. 由補給車載送單車及人員返回龍潭大池

## 十五、單車檢查建議

活動前請先完成基本車況檢查。

| 項目 | 檢查重點 |
|---|---|
| 煞車 | 確認前後煞車正常，煞車皮未過度磨損 |
| 輪胎 | 確認胎壓足夠，外胎沒有明顯裂痕、破損或異物 |
| 變速 | 確認前後變速順暢，避免上坡時無法順利換檔 |
| 鏈條 | 確認鏈條潤滑正常，沒有異音或卡頓 |
| 車燈 | 確認前後車燈可正常開啟，電量充足 |

## 十六、團騎規則

為確保所有參加者安全，請遵守以下團騎規則：

1. 聽從領騎、押隊及交管人員指示。
2. 不任意超越領騎。
3. 不落後押隊太遠。
4. 騎乘時保持安全車距。
5. 不突然煞車或變換方向。
6. 需要停車時，請先靠邊並提醒後方。
7. 發現其他隊員有狀況，請協助通知工作人員。
8. 補給點集合後，依工作人員指示再出發。
