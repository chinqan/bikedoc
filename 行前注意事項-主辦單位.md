## 二、各組負責事項

### 1. 總教練 — 帶操
- 07:20 帶領全體進行暖身操
- 暖身內容：肩、腰、膝、踝關節活動 + 拉筋
- 結束後配合金錢豹進行團體拍照

### 2. 領隊 — 全程指揮與管控

| 任務 | 內容／要點 |
|---|---|
| 🚦 **交管** | 確認交管人員就定位、車隊聽從指示通過。地點：**大北坑（1 km）**、**內環山（8.3 km）**、各補給點前後、折返點、加油站進出口、回程接近龍潭大池路段。<br>⚠️ 補給車較難臨停，**出發前請指派一位鐵爸／鐵媽擔任押隊**，確認全組通過後再前進。 |
| 🚧 **施工路段** | 提前廣播提醒：全隊**減速慢行**、**單線依序通過**；必要時下車**牽車**通過；雨天路面濕滑須加倍謹慎。<br>⚠️ 補給車同樣難以臨停，押隊鐵爸／鐵媽務必殿後，確認全組通過。 |
| 💧 **補水提醒** | 每個補給點到站時主動提醒「補水、檢查水量」，並關注隊員臉色與狀態。 |
| 🔄 **折返點**（7-11 騰達門市，20 km） | 依工作人員指示停靠、不佔用店家出入口；確認全隊到齊、補水、上廁所完成後再折返；折返時注意對向來車與隊伍方向。 |
| ⏱ **休息時間控管** | 每個補給點停留**控制在 10～15 分鐘**，避免拖延：<br>　• 08:10–08:20　補給1：國道三號路橋下<br>　• 09:20–09:30　折返：7-11 騰達門市<br>　• 10:00–10:10　補給2：濟世宮<br>　• 10:40–10:50　補給3：中油正新站 |

## 六、雨天備案

- 主辦單位於活動前一日（5/23）晚間 20:00 前依氣象預報決定是否照常舉行，並於 LINE 群組統一公告。
- 若照常舉行：
  - 領騎特別注意**控速**，視情況決定是否於施工路段牽車通過
  - 全員務必攜帶**雨具**（雨衣／防水袋）
  - 補給車多備毛巾與乾衣物
  - 若遇大雨、路況不佳，**領隊有權調整路線或部分路段棄賽**
- 提醒參加者：可安全行進下盡力完成，但**安全優先**。
